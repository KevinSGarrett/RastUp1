B\) Layout Patterns

Page shell: Sticky header; 12column grid at MD+; single column at SM.
Side drawer for filters on desktop; fullscreen on mobile.

Sticky bars: Search (Filters/Save), Profile/Studio (Book/Request),
Checkout (Pay & Confirm).

Cards first: All lists (search, boards, invites) render cards with
consistent skeletons.

C\) Global Components (catalog)

1\) Cards

PersonCard(role) --- for People
(Model/Photographer/Videographer/Creator/FanSub)

Top: cover image/video thumbnail (3:2 crop), Role chip, Location chip

Meta row: Name, Reputation ring (0--100 with tooltip "Why this score?"),
price band, distance

Chips: Verified (ID), Trusted Pro (if BG), Instant Book, Has/Owns Studio

Social chips: IG/YT/TT/X (followers; ER% if verified connection)

Actions: View, Message, Add to Board; Promoted label (when applicable)

StudioCard --- for Places

Top: hero image (3:2), Neighborhood, Price/hr, Min hours

Chips: Verified Studio, Instant Book, amenity set (Natural light • Cyc
wall • Blackout • 14' ceilings • Parking)

Meta: Capacity, rating ★ count, host card hover

Actions: View, Add to Board; Promoted label (when applicable)

BoardCard

Works with mixed types (People + Studios); shows type icon, status chip
(Invited/Quoted/Awarded/Booked).

CaseStudyCard

Before/after thumbnails, tags, city; links to featured People/Studios.

SkeletonCard (both types)

Media gray box + 2--3 text bars + placeholder chips; shimmer at 60fps.

2\) Chips & Badges

Role chip (Model/Photographer/...) --- colorcoded

Verified (ID) --- checkmark chip

Trusted Pro --- shield chip

Verified Studio --- building/check chip

Instant Book --- ⚡ chip

Safe Mode gated --- subtle lock on nuditytier chip (only visible to
owner)

Nudity tier --- None/Lingerie/Artistic; SafeModeaware (never shown in
emails)

Social metric chips --- IG 182k • 2.8% ER, YT 210k • 45k avg views, TT
530k • 8.1% ER

Promotion chip --- Promoted (always labeled; never used as rank
override)

3\) Filters & Search

SearchBar

Role/Listing selector tabs (People vs Studio divider), query field, city
chip, date/time pill

FilterDrawer

Common: city/radius, availability window, price/budget, distance, min
Rep, Verified/Trusted toggles

Peoplespecific:

Models/Creators: look & comfort (height, hair, measurements, Nudity
tier, chaperone policy)

Photographers/Videographers: specialties, deliverables, turnaround

FanSub Creators: platform links, promo packs, posting window/minlive
controls

Studiospecific: price/hr, min hours, capacity, size, amenities (chips),
Instant Book, Verified Studio

Social Metrics panel: perplatform toggles; simple Engagement slider
(Low→High) + Advanced (ER ≥ %, Avg views ≥ N); ANY/ALL platform logic;
Verifiedonly toggle

Saved Search bar: Save (cadence dropdown: off/daily/weekly), name field
