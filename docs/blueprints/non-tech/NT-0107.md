I\) Money Movement & Docs

Refunds: autoissue to original method; partials itemized; tax
recalculated; buyer fee refunded per rules.

Adjustments (studio): create new charge from escrow (if available) or
postcharge with receipt.

Deposit capture: separate, up to cap; no platform fee; receipt with
evidence link.

Amended docs: SOW/House Rules/Release updates autoissued on material
change orders.

J\) Special Cases & Guards

UGC/promo noncompliance: if post removed before minlive, the seller gets
one retry; else partial refund per package policy.

Expired external links: seller has 48h to refresh; timer pauses;
repeated expiry → reputation penalty.

Fraud guard: repeated loss in disputes → IB off and promotion
suppression; investigation flag to Safety.

Chargebacks: we do not charge a dispute fee unless clear abuse;
otherwise platform absorbs processor fee.
