Funnel: search appearances → profile views → messages → quotes sent →
awards → bookings → ontime delivery → reviews.

daypart, city radius.

X%" or "add lowertier package."

100k+").

Availability utilization: % of IB windows filled; value of lastminute
fills.

Boost performance (if used): impressions, clicks, CPC, inquiries,
bookings, incremental lift (holdout).

Quality & disputes: revision rate, dispute rate, average cure time.

N\) KPIs & Targets (first 90 days / city)

Activation: ≥ 50% of new sellers publish ≥1 package within 7 days.

Responsiveness: median first response ≤ 2h; \>80% of invites answered
within 24h.

Conversion: view → message ≥ 12%; message → award ≥ 30%; award →
completed ≥ 90%.

Utilization: Instant Book adoption ≥ 40% of active sellers; lastminute
fill used by ≥ 20%.

Delivery health: autoaccept ≥ 50%; deliverable disputes \< 5%; revision
cycles avg ≤ 1.3.

Reputation: avg Rep Score ≥ 78 by day 60 for active cohort.

Economics: Early Payout usage 10--20% among eligible; boost ROAS (for
users who try it) ≥ 2.5×.
